Does Price Affect Satisfaction?

Introduction

This project explores how price affects brand rank and satisfaction.

Does the price difference between brands influence customer satisfaction rankings?
In this study, we analyzed data comparing luxury brands and low-cost brands to understand how pricing affects consumer perceptions and overall satisfaction.

Stepwise Findings

We start with hypothesis testing and move on to regression analysis.

Dataset Description

This project is based on a dataset of 100 fashion items, equally divided between luxury brands and affordable brands (50 samples each). The goal is to explore whether price influences customer satisfaction, as measured by product rankings.

The data was manually collected from an online fashion platform and includes the following variables for each item:

  • Brand ‚Äì The name of the fashion brand (e.g., Gucci, Zara)
  • Product Name ‚Äì Specific item or model name
  • Price ‚Äì Retail price in USD
  • Ranking ‚Äì User rating score (1‚Äì5)
  • Category ‚Äì Product type

We intentionally balanced the dataset to include both premium and budget segments, allowing a fair comparison across similar product categories.

Regression Models: Understanding Price & Satisfaction

We tested four types of regression models to analyze how price affects product ranking (satisfaction):

1. Original Model

Equation: Ranking = Œ≤‚ÇÄ + Œ≤‚ÇÅ¬∑Price

This model shows the total effect of price on satisfaction. If Œ≤‚ÇÅ > 0, then higher-priced items tend to get higher rankings.

2. Confounder Model

Equation: Ranking = Œ≤‚ÇÄ + Œ≤‚ÇÅ¬∑Price + Œ≤‚ÇÇ¬∑Category

Category might affect both price and ranking. Including it helps remove bias in Œ≤‚ÇÅ. The difference in Œ≤‚ÇÅ (before and after) tells us how much the confounder matters.

3. Moderator Model

Equation:Ranking = Œ≤‚ÇÄ + Œ≤‚ÇÅ¬∑Price + Œ≤‚ÇÇ¬∑BrandType + Œ≤‚ÇÉ¬∑(Price √ó BrandType)

This model checks if the price effect changes depending on the brand type (Luxury vs. Affordable). If Œ≤‚ÇÉ is positive, price has a stronger effect for luxury brands.

4. Mediator Model

Equation: Ranking = Œ≤‚ÇÄ + Œ≤‚ÇÅ¬∑Price + Œ≤‚ÇÇ¬∑Popularity

If popularity is on the path from price to ranking, this model shows only the direct effect of price. The drop in Œ≤‚ÇÅ indicates the size of the indirect (mediated) effect.
