Is The More Expensive Always Better?

Introduction

This project explores how price affects brand rank and satisfaction.

Does the price difference between brands influence customer satisfaction rankings?
In this study, we analyzed data by looking at the price and customer ranking to understand how pricing affects consumer perceptions and overall satisfaction.

Stepwise Findings

We start with hypothesis testing and move on to regression analysis.

Dataset Description

This project is based on a dataset of 1472 cosmetic items with their price, ranking, and also, the division of luxury brands and non-luxury brands . The goal is to explore whether price influences customer satisfaction, as measured by product rankings.

The dataset was obtained from a public online cosmetics platform and includes detailed product-level information. The variables are as follows:

Variable Description
Name The product name or label (e.g., Cr√®me de, Facial Treatment)
Brand The name of the cosmetic brand (e.g., LA MER, SK-II, DRUNK ELEPHANT)
Price Retail price of the product (numeric value, in USD)
Rank User rating score based on customer reviews (scale: 1 to 5)
Skin Type Binary variables (0/1) indicating whether the product is suitable for each skin type (Dry, Normal, Oily, Sensitive)
Brand Type Categorical variable classifying the brand as either "Luxury" or "Non-luxury"

We intentionally balanced the dataset to include both premium and budget segments, allowing a fair comparison across similar product categories.

Model 2: Confounder Model


# Equation:
# Ranking = Œ≤‚ÇÄ + Œ≤‚ÇÅ¬∑Price + Œ≤‚ÇÇ¬∑Category

# Interpretation:
# 'Category' affects both Price and Satisfaction (Ranking), so it's a confounder.
# Including it removes bias in Œ≤‚ÇÅ (the effect of Price on Ranking).
# The difference in Œ≤‚ÇÅ before and after adding Category shows the confounding strength.

Model 3: Moderator Model


# Equation:
# Ranking = Œ≤‚ÇÄ + Œ≤‚ÇÅ¬∑Price + Œ≤‚ÇÇ¬∑Normal + Œ≤‚ÇÉ¬∑(Price √ó Normal)

# Interpretation:
# 'Normal' is a moderator ‚Äî it changes the strength of Price‚Äôs effect on Ranking.
# If Œ≤‚ÇÉ > 0, the influence of Price on Ranking is stronger when Normal is higher.
# For example, luxury brands may show stronger effects than everyday brands.

Model 4: Mediator Model


# Equation:
# Ranking = Œ≤‚ÇÄ + Œ≤‚ÇÅ¬∑Price + Œ≤‚ÇÇ¬∑Popularity

# Interpretation:
# If Price ‚Üí Popularity ‚Üí Ranking, then Popularity is a mediator.
# This model isolates the direct effect of Price.
# A large drop in Œ≤‚ÇÅ (compared to Model 1) suggests a strong indirect effect via Popularity.
